# step 06

## Motion

```
^

  [N] Move to first non-blank character of line.

W B

  [N] Like w and b, but moves by WORD (any sequence of non-blank characters).
      See `:help word` and `:help WORD`

e E

  [N] Move to end of the current word or WORD.
```

## Navigation

```
C-d C-u

  [N] Move down a half-page, move up a half-page. Cursor position will change.
      Compare to C-f C-b (page navigation) and C-e C-y (scrolling).
```

## Settings

```
:set list
:set nolist

  [C] Enable/disable 'list' mode, which reveals non-visible characters (e.g., tabs vs spaces).
      See `:help 'list'`
```
